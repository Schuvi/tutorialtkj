Tentang Website

Apa alasan saya membuat website ini?

Saya membuat website ini, karena saya melihat teman-teman saya kesulitan dalam mencari tutorial, ada beberapa sumber tutorial tetapi terkadang tutorial tersebut kurang jelas, sulit dipahami, dan itu pun menggunakan versi yang kurang update.

Tujuan website ini dibuat?

Tujuannya untuk membantu teman-teman saya agar mendapatkan tutorial yang sesuai dengan apa yang mereka perlukan, dengan tutorial yang jelas dan tidak membingungkan.

Salam Hangat Dari Saya

Untuk teman-teman ataupun adik-adik yang nantinya menggunakan website saya untuk tutorial, semangat belajarnya karena sesuatu itu tidak didapatkan secara instan tetapi dengan kerja keras.

Awalnya saya juga tidak bisa, tapi karena saya suka dan mau berusaha akhirnya bisa. Dunia Informatika itu tidak hanya tentang koding, jaringan, dll. Tetapi Informatika itu tentang pemahaman, logika, dan problem solving.
